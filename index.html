3D modeling is the process of creating a virtual model of your home with all details
Such 3D model allows you to see the design project before the start of the repair so that you can assess in advance whether such a solution suits you

You will need:
- All drawings in pdf format
- Furniture layout plan
- Ceiling plan with lights location
- Sweeps on the walls with a description of the finish, the layout of finishing materials, indicated where to use them.
- Floor plan with a detailed description of the flooring and laying direction
- A selection of furniture, lighting, plumbing, large decorative elements (names, articles, color or link to this item)
- References of interiors, furniture, moodboards or collages
